# Swatch Internet Time: Vanilla JS PWA

version 0.1.0 

A plain vanilla Javascript PWA app to display the current Swatch Internet Time. This is the "no frills" version that just displays the current .beat time with no additional features. There is a newer version created in Preact which offers more features. See this version online at: https://kendawson.online/free/swatch/
